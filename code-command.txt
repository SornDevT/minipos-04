ຄຳສັ່ງໃນ Laravel + vue

- composer install                                  = ຄຳສັ່ງຕິດຕັ້ງ composer ໃນໂປຼເຈັກຂອງ Laravel
- npm install                                       = ຄຳສັ່ງຕິດຕັ້ງ Node JS Packet
- php artisan serve                                 = ຄຳສັ່ງ Run ໂປຼເຈັກ Laravel
- php artisan route:cache                           = ເຄຼຍ Router ກໍລະນີ post ຂໍ້ມູນບໍ່ເຫັນ
- npm run dev                                       = ຄຳສັ່ງ Run Compli Laravel
- npm run watch                                     = ຄຳສັ່ງ Run Compli Laravel ແລະ ຕິດຕາມການປ່ຽນແປງ
- php artisan make:controller ປ້ອງຊື່ controller       = ຄຳສັ່ງສ້າງ Controller
- php artisan make:model -m ປ້ອນຊື່ model              = ຄຳສັ່ງສ້າງ Model
